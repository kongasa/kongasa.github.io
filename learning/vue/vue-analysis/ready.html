# 基础准备

# Flow

JavaScript 的静态分析工具，在未运行时检查出代码缺陷

Flow 的工作方式主要有两种。类型推断和类型注释。类型注释需要在写代码时进行一些类似 TypeScript 中的类型声明。

Flow 是 Vue v2.x 采用的静态分析工具。在 Vue v3.x 中使用的是 TypeScript。
